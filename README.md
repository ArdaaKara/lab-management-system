# Computer Lab Management System (CLMS)

Eğitim kurumlarındaki bilgisayar laboratuvarlarının donanım envanterini ve arıza yönetim süreçlerini yöneten hafif IT bakım sistemi.

## Teknoloji Stack
- **Backend:** Java 21, Spring Boot 3.3, MySQL 8.0
- **Frontend:** React 19, Vite, Tailwind CSS
- **Altyapı:** Docker, docker-compose

## Kurulum
```bash
cp .env.example .env
docker compose up -d
```
